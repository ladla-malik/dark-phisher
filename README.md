##

### Installation

- Just, Clone this repository -
  ```
  git clone --depth=1 https://github.com/ladla-malik/dark-phisher.git
  ```

- Now go to cloned directory and run `zphisher.sh` -
  ```
  $ cd dark-phisher
  $ bash darkphisher.sh
  ```

- On first launch, It'll install the dependencies and that's it. ***darkphisher*** is installed.

- Install the `.deb` file by executing

  Or
  ```
  $ dpkg -i <your path to deb file>
  $ apt install -f
  ```

##

### Run on Docker

- Docker Image Mirror:
  - **DockerHub** : 
    ```
    docker pull ladla-malik/dark-phisher
    ```
  - **GHCR** : 
    ```
    docker pull ghcr.io/ladla-malik/dark-phisher:latest
    ```

- By using the wrapper script [**run-docker.sh**](https://raw.githubusercontent.com/htr-tech/zphisher/master/run-docker.sh)

  ```
  $ curl -LO https://raw.githubusercontent.com/ladla-malik/dark-phisher/master/run-docker.sh
  $ bash run-docker.sh
  ```
- Temporary Container

  ```
  docker run --rm -ti ladla-malik/dark-phisher
  ```
  - Remember to mount the `auth` directory.
